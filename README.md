# Task Movies

Modern movie search application built with React

## Tech Stack

- React
- TanStack Query
- Axios
- Tailwind CSS

## Features

- Movie Search
- Responsive Design
- API Integration
- Loading States
- Error Handling
- Dynamic Rendering

## Installation

```bash
npm install
npm run dev
```
